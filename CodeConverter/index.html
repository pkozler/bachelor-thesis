Automatický převod kódu

Tento nástroj slouží jako pomůcka pro převod zdrojového kódu jazyka Java do zdrojového kódu zvoleného jazyka využívajícího knihovny s definovaným rozhraním v duchu jazyka Java.
Nástroj provádí automatický částečný převod některých odlišně vypadajících konstrukcí pomocí regulárních výrazů.
Upozornění: jsou prováděny pouze hrubé základní úpravy bez komplexní analýzy, není produkován finální kód!
